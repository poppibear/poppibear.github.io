地铁惊魂

发表于 2019-06-23 更新于 2020-07-18 分类于 故事

周五下班，地铁返家，女同事下车后空出了座位，体大难负的我也没啥客气的就坐下了。打开油管，卡顿中有滋有味地看着所谓千人千面算出来的台湾政论(瞎扯)，就压根没理会到惠新西街站上来后站我跟前的人。

可能是用了reno十倍变焦手机，自己的眼睛好像也能十倍了，手机越放越低，丝毫没有模糊(也可能是屏幕够大)。这时腿好像被蚊子叮了，奇痒难耐，下意识的就拿着手机去挠(同胞的节目吗，只靠声音表情也能演绎的山重水复十八弯，完全不影响观看体验)。

挠痒这件事大家懂得，只能曲中求，不能直中得，好容易找对了位置，正挠的百转千回，眼角余光突然瞄到了蕾丝似的东西——“完了”，心里一咯噔，高度政治敏感的我意识到了问题的严重性。心理祷祝：智慧，如果有的话，是该你出场的时候了。

心念速转，眼前有两条路，一条是装作不知道，继续挠(奇怪的是它现在反而倒不痒了); 一条是主动上前解释，把爱机与人，自证清白(说有第三条路赶紧收起手机当一切没发生的，得赶紧报班补下基础教育了)。

然而毕竟不在年轻，智商也行动缓慢，转念见蕾丝裙已然退后了两步，想自证清白也经失去了好时机，只能继续假装挠着快出血的腿，再继续假装低头看着视频。

估计那一刻我可能已经失明了，因为把所有的感知能力集中在了听觉上，而且是屏蔽了蓝牙耳机的声音后聚焦在身前两步的地方——蕾丝裙在和闺蜜窸窸窣窣，估计在做出最后的裁决，此刻的我像是占了水的拖把，只能任人摁在地上摩擦。我调慢呼吸，面对命运，静静地等待着一切的发生(奇怪的是还有种莫名的期待，抖M得很)。

然而什么都没有发生，快到三里屯时两个女生开始朝车门移转，我才敢直起腰来，后背已然湿了大片，我的策略终于成功了——我这样想着摘下耳机，不知道为何耳朵被震得疼——又或许是我人畜无害的形象拯救了我，然而事实告诉我我想多了，正准备下车的蕾丝裙的背影依然散发着怀疑的姿态，然后旁边的闺蜜用世界上最亲切的北方口音赦免了我：瞎想啥呢，人家看的是新闻联播！

一脸黑人问号的我看向屏幕，原来节目早自动跳到了习大大访朝的画面，视频中朝鲜人民举行着盛大的欢迎仪式，齐刷刷的掌声久久未停，难怪我耳朵生疼。

朝鲜三大乐团的音乐家唱起了我们国家的歌曲，当时当刻，那充满情感的歌声让自己如此动容，感谢朝鲜人民，感谢中朝人民的伟大友谊，感谢社会主义革命情感拯救了差点“被猥琐”的我。
